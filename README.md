# 🚁 Helicopter Simulator Private Project

A collaborative engineering project by **Jacob Mueller and Paco Maranhao**, focused on designing and building a fully customizable, 3D-printable helicopter simulator rig inspired by the legendary **Bell UH-1 “Huey”**.

The goal is to recreate realistic helicopter controls and cockpit interaction using accessible tools, off-the-shelf electronics, and 3D printing.

Note that the Branch "main" is not at all up to date and needs to be merged from various dev branches. This project is a work in progress.

---

## 🎯 Project Goal

- Build a fully functional helicopter simulator rig
- Replicate UH-1 Huey control layout as closely as possible
- Design a modular and customizable cockpit system
- Ensure all parts are 3D-printable
- Use easily available electronics (e.g., Amazon components)
- Integrate with PC flight simulation software

---

## 🧠 Design Philosophy

- **Realism-focused** – authentic control feel and layout
- **Modular design** – easy upgrades and replacements
- **Accessible build** – consumer-grade tools and parts
- **Customizable** – adaptable to different simulator setups

---

## 🛠️ Tools & Technologies

- **CAD Software:** Siemens NX  
- **Fabrication:** FDM 3D Printing  
- **Electronics:** Teensy 4.1 / USB HID controllers / Hall-Sensors

## 🧩 Planned Components
